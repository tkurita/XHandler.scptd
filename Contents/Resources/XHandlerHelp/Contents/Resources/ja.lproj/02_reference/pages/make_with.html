make_with

make_with(handler_name, nargs)

ハンドラの名前と引数の数を指定して、XHandler のインスタンスを生成します。

XHandler のインスタンスは指定したハンドラを実行する為のスクリプトオブジェクトです。実行するハンドラは位置パラメータを持つことができます。ラベルパラメータを持ったハンドラは扱えません。

Parameters
  • handler_name (text) : ハンドラの名前
  • nargs (integer) : 呼び出すハンドラがとる引数の数
Result

script object
