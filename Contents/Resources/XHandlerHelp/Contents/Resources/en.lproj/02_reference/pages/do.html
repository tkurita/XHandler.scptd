do

do(target_object, variable_arguments)

Call the specified handelr of a passed script object (target object) with a passed arument.

Parameters
  • target_object (script object) : a script object of which handler is called.
  • variable_arguments : arguments passed to the handler. Pass arguments of numbers specfied with nargs in make_with as arg1, arg2, ...
Result

: a returned value from the handler.
